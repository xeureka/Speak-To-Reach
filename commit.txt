feat: full-stack education management system with Tailwind, NeonDB, and role-based access

Backend:
- Migrate from in-memory to PostgreSQL (NeonDB) with Drizzle ORM
- Auth: JWT login, password change, teacher password reset (admin)
- CRUD: teachers, students, courses, sections, enrollments, class sessions
- Excel import for bulk student creation (XLSX parsing)
- Session attendance (per-student) and reports (per-session) with upsert
- Recurring schedule bulk session creation (generate weeks of sessions at once)
- Teacher activity log for payment tracking
- Reports analytics endpoint (low attendance, missing reports, behind schedule)
- Teacher-specific endpoints: assigned students, teaching analytics
- Dashboard aggregation for admin and teacher views

- React 19 + TanStack Router/Query + Hono OpenAPI client
- Admin dashboard: 4 metric cards, active sections, today's classes
- Teacher dashboard: sections, today's schedule, completed count

- Teacher detail page: analytics (sections, students, monthly stats, reports), horizontal session cards, private/group course breakdown
- Students page: Excel import with drag/drop, admin-only create/import
- Teacher students page: read-only view of assigned students only
- Reports page: analytics with teacher missing-reports alert
- Payments page: monthly audit by teacher
- Sidebar: animated collapse/expand, profile dropdown with portal, role-based avatars, online indicator
